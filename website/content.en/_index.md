# JVP - Virtualization Platform

**JVP** (jimyag's virtualization platform) is a virtualization platform based on QEMU/KVM and libvirt, providing complete virtual machine lifecycle management.

{{% columns %}}
- ### Simple
  Deploy with Docker in minutes. Zero configuration required to get started.

  ```bash
  docker compose up -d
  ```

- ### Powerful
  Full VM lifecycle management: create, start, stop, snapshot, template, and more.

- ### Modern
  Beautiful web UI built with React. VNC and Serial console included.
