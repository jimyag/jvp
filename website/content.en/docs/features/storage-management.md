# Storage Management

Manage storage pools and volumes by node.

## Storage Pools

- List storage pools by node
- Create new storage pools
- Start and stop storage pools
- Delete storage pools
- View storage pool usage statistics

## Storage Volumes

- List volumes by node and storage pool
- Create new volumes
- View volume details:
  - Capacity
  - Allocation
  - Format
- Delete volumes
