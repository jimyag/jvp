# JVP Documentation

Welcome to the JVP documentation. JVP is a virtualization platform based on QEMU/KVM and libvirt.

## Features

- **Instance Management** - Create, start, stop, reboot, delete VMs with cloud-init support
- **Multi-Node Support** - Manage multiple libvirt nodes (local and remote)
- **Storage Management** - Manage storage pools and volumes
- **Snapshot Management** - Create, revert, and delete snapshots
- **Template Management** - Register and manage VM templates
- **Key Pair Management** - SSH key pair management with auto-injection
- **Web UI** - Modern React-based management interface
- **Remote Console** - VNC and Serial console support
