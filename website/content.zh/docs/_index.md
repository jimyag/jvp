# JVP 文档

欢迎阅读 JVP 文档。JVP 是一个基于 QEMU/KVM 和 libvirt 的虚拟化平台。

## 功能特性

- **实例管理** - 创建、启动、停止、重启、删除虚拟机，支持 cloud-init
- **多节点支持** - 管理多个 libvirt 节点（本地和远程）
- **存储管理** - 管理存储池和存储卷
- **快照管理** - 创建、恢复和删除快照
- **模板管理** - 注册和管理虚拟机模板
- **密钥对管理** - SSH 密钥对管理，支持自动注入
- **Web 界面** - 现代化的 React 管理界面
- **远程控制台** - 支持 VNC 和串口控制台
